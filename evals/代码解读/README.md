# DeepAgents Evals 源码解读

> 基于 `deepagentsjs/evals/` 目录源码分析，目标是为构建类似行为评估框架提供完整参考。

## 项目简介

`evals/` 是 DeepAgents 的**行为评估套件集合**，用于对 AI Agent 进行系统化的能力测试。它不测试代码逻辑（那是单元测试的职责），而是测试**Agent 在真实 LLM 推理下的行为是否符合预期**。

- **测试框架**：[Vitest](https://vitest.dev/) + [LangSmith](https://langsmith.com/) 实验追踪
- **运行方式**：设置 `EVAL_RUNNER` 环境变量选择模型，运行 `vitest run`
- **套件数量**：17 个独立评估套件，覆盖从基础到专业的各类 Agent 能力
- **基础设施**：`internal/eval-harness/` 提供共享的 Runner、断言工具和模型注册
