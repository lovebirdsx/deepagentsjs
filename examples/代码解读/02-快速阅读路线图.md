## 按"我想实现什么"导航

### 我想搭一个能上网搜索并写报告的 Agent

1. [研究型 Agent](./03-研究型Agent.md) — 看主 Agent 如何将任务拆给研究子代理和评审子代理
2. [异步子代理并行研究](./07-异步子代理并行研究.md) — 看如何用多个研究者同时并行调查不同角度

### 我想让 Agent 记住项目规范/代码风格

1. [记忆与上下文 Agent](./04-记忆与上下文Agent.md) — AGENTS.md 文件注入，Agent 启动即可读取
2. [技能与记忆 Agent](./05-技能与记忆Agent.md) — AgentMemoryMiddleware，支持长期记忆写入

### 我想让 Agent 读写文件、操作磁盘

1. [后端对比与选型](./10-后端对比与选型.md) — FilesystemBackend vs StateBackend 的差异
2. [沙箱执行环境](./11-沙箱执行环境.md) — 如果需要隔离执行，看各沙箱选型

### 我想让 Agent 执行代码（Shell 命令/Python/Node.js）

1. [沙箱执行环境](./11-沙箱执行环境.md) — 6 种沙箱：云端（Daytona/Deno/Modal）、本地（LocalShell/VFS）
2. [REPL 与 RLM 模式](./13-REPL与RLM模式.md) — QuickJS REPL 做数学计算和数据变换

### 我想实时显示 Agent 的思考过程

1. [流式输出完全指南](./12-流式输出完全指南.md) — 8 个 streaming 示例，从 token 流到自定义事件

### 我想构建多层级的 Agent 系统（主 Agent → 专家 Agent → ...）

1. [层级嵌套 Agent](./06-层级嵌套Agent.md) — 3 层嵌套示例，DeepAgent 作为子代理
2. [自托管异步子代理服务器](./08-自托管异步子代理服务器.md) — 把子代理部署成独立 HTTP 服务

### 我想把 Agent 集成到 IDE（Zed/JetBrains）

1. [ACP 服务器集成](./09-ACP服务器集成.md) — DeepAgentsServer 一行启动

### 我想为 Agent 添加可插拔的能力（技能）

1. [技能文件解读](./14-技能文件解读.md) — SKILL.md 格式、4 个真实技能的内容分析
