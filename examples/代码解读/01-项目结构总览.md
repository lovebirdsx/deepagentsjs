# 项目结构总览

`examples/` 是 deepagents 框架的官方示例目录，以 pnpm workspace 方式组织，与库本体共享类型定义和构建产物。

---

## 目录树

```
examples/
│
├── package.json                    # examples workspace 包配置
├── tsconfig.json                   # TypeScript 配置（继承 monorepo 根配置）
│
├── research/                       # 研究型 Agent（主 + 子代理）
│   ├── research-agent.ts           # 核心：研究 + 评审双子代理
│   ├── langgraph.json              # LangGraph CLI 部署配置
│   └── .env.example
│
├── memory/                         # AGENTS.md 记忆注入示例
│   ├── memory-agent.ts             # Agent 创建 + FilesystemBackend + memory 参数
│   └── AGENTS.md                   # 示例记忆文件（项目上下文）
│
├── skills-memory/                  # 技能 + AgentMemory 综合示例
│   └── skills-memory-agent.ts      # 动态创建临时 SKILL.md + AgentMemoryMiddleware
│
├── hierarchical/                   # 层级嵌套 Agent
│   └── hierarchical-agent.ts       # 3 层嵌套：主 → 研究专家（DeepAgent）→ 事实核查子代理
│
├── async-subagents/
│   └── parallel-research/          # 并行异步研究（LangGraph Platform + React UI）
│       ├── supervisor.ts           # 督导 Agent，用 AsyncSubAgent 扇出
│       ├── researcher.ts           # 研究者子图，含 CompletionCallback 中间件
│       ├── langgraph.json          # 注册 supervisor + researcher 两个图
│       └── ui/                     # Vite + React 状态展示前端
│           ├── src/App.tsx
│           ├── src/components/ResearcherCard.tsx
│           └── src/types.ts
│
├── async-subagent-server/          # 自托管 Agent Protocol 服务器
│   ├── server.ts                   # Hono + PostgreSQL 实现 6 个 Agent Protocol 端点
│   ├── supervisor.ts               # 与服务器对接的督导 Agent REPL
│   ├── Dockerfile
│   └── docker-compose.yml
│
├── acp-server/                     # ACP（Agent Client Protocol）服务器
│   └── server.ts                   # DeepAgentsServer，供 IDE（Zed/JetBrains）接入
│
├── backends/                       # 后端选型示例（5 个文件各演示一种后端）
│   ├── state-backend.ts            # StateBackend（默认，存于 LangGraph 状态）
│   ├── store-backend.ts            # StoreBackend（跨线程持久化）
│   ├── filesystem-backend.ts       # FilesystemBackend（写入本地磁盘）
│   ├── composite-backend.ts        # CompositeBackend（按路径前缀路由多后端）
│   └── local-shell-backend.ts      # LocalShellBackend（文件系统 + Shell 执行）
│
├── sandbox/                        # 各种沙箱后端示例（6 个文件）
│   ├── daytona-sandbox.ts          # Daytona 云沙箱
│   ├── deno-sandbox.ts             # Deno Deploy 微型虚拟机
│   ├── modal-sandbox.ts            # Modal 容器沙箱
│   ├── vfs-sandbox.ts              # Node.js 内存虚拟文件系统
│   ├── langsmith-sandbox.ts        # LangSmith 云沙箱（内置于 deepagents）
│   └── local-sandbox.ts            # 自定义 BaseSandbox 实现（LocalShellSandbox）
│
├── streaming/                      # 流式输出示例（8 个文件）
│   ├── basic.ts                    # 基础子图流
│   ├── tokens.ts                   # 逐 token 流（messages 模式）
│   ├── tool-calls.ts               # 工具调用流（AIMessageChunk 解析）
│   ├── progress.ts                 # 步骤进度流（updates 模式）
│   ├── lifecycle.ts                # 子代理生命周期追踪
│   ├── filter-by-type.ts           # 按来源（主 Agent / 子代理）分路由
│   ├── custom-updates.ts           # 自定义进度事件（config.writer）
│   └── multi-mode.ts               # 多模式组合（updates + messages + custom）
│
├── repl/                           # REPL 代码执行示例
│   ├── data-analysis-agent.ts      # QuickJS REPL：数据分析 + 文件读写
│   └── rlm-agent.ts                # RLM 模式：REPL 中通过 PTC 并行调用子代理
│
└── skills/                         # 技能文件示例（SKILL.md 格式）
    ├── arxiv-search/
    │   ├── SKILL.md                # arXiv 论文搜索技能描述
    │   └── arxiv_search.ts         # 技能对应的可执行脚本
    ├── web-research/
    │   └── SKILL.md                # 结构化网络研究工作流
    ├── file-organizer/
    │   └── SKILL.md                # 文件组织技能
    ├── langgraph-docs/
    │   └── SKILL.md                # LangGraph 文档检索技能
    └── skill-creator/
        ├── SKILL.md                # 技能创建指南（元技能）
        └── scripts/
            ├── init_skill.ts       # 初始化技能目录脚手架
            └── quick_validate.ts   # 技能格式验证脚本
```

## 各子目录定位一览

| 子目录                   | 主题           | 复杂度 | 适合学习   |
| ------------------------ | -------------- | ------ | ---------- |
| `memory/`                | 记忆注入       | ★☆☆☆   | 入门       |
| `backends/`              | 存储后端       | ★★☆☆   | 架构决策   |
| `streaming/`             | 流式输出       | ★★☆☆   | 用户体验   |
| `research/`              | 子代理编排     | ★★★☆   | 复杂 Agent |
| `hierarchical/`          | 嵌套代理       | ★★★☆   | 多级系统   |
| `sandbox/`               | 代码执行       | ★★★☆   | 安全隔离   |
| `repl/`                  | REPL + RLM     | ★★★☆   | 极致并行   |
| `async-subagents/`       | 异步远程子代理 | ★★★☆   | 分布式     |
| `async-subagent-server/` | 自托管服务器   | ★★★★   | 生产部署   |
| `acp-server/`            | IDE 集成       | ★★☆☆   | 工具集成   |
| `skills/`                | 技能文件       | ★☆☆☆   | 能力扩展   |
