# Examples 源码解读文档

> 基于 `deepagentsjs/examples` 目录下的全部源码，目标是为构建类似 AI Agent 系统提供完整参考。

## 目录说明

`examples/` 是 deepagents 框架的**官方示例集合**，覆盖了从最简用法到生产级部署的各种场景，包括：

- 不同类型的 Agent 构建模式（研究型、编码型、协作型）
- 各种存储后端（状态内存、持久化 Store、文件系统、云沙箱）
- 子代理编排（同步、异步、层级嵌套）
- 流式输出的各种消费方式
- 技能（Skill）与记忆（Memory）集成
