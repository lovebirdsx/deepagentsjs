## 关键代码（server.ts:352-419）

```typescript
private async handleInitialize(params): Promise<InitializeResponse> {
  // 1. 提取客户端信息
  const clientName = clientInfo?.name ?? params.clientName ?? "unknown"

  // 2. 存储客户端能力
  this.clientCapabilities = {
    fsReadTextFile: fs?.readTextFile ?? false,
    fsWriteTextFile: fs?.writeTextFile ?? false,
    terminal: clientCaps.terminal !== undefined,
  }

  // 3. 版本协商（接受客户端版本）
  const requestedVersion = parseInt(String(params.protocolVersion), 10) || 1

  return {
    protocolVersion: requestedVersion,
    agentInfo: { name: this.serverName, version: this.serverVersion },
    agentCapabilities: { ... },
    authMethods: this.authMethods,
  }
}
```

## 注意事项

- `protocolVersion` 必须是 `number` 类型（ACP 规范要求），代码做了显式解析
- `authenticate` 方法是 no-op（直接返回），当前版本不做认证校验
- 握手只在连接初始时进行一次，能力标记在整个连接生命周期内不变
